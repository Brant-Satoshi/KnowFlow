## Verdict

Hybrid adds no hit-rate or Recall@5 benefit on this corpus. Without rerank, Recall@1 and MRR improve but Precision@5 and nDCG@3 decline, so the raw-ranking signal is mixed rather than a consistent lift. Production-style rerank erases almost all quality differences, while this point-in-time run measured higher average/p50/p95 latency. Keep `HYBRID_SEARCH_ENABLED` off by default and treat hybrid as an experiment until a keyword-heavy target corpus shows a repeatable quality gain that justifies its cost.

## Reproduce

```bash
pnpm seed:demo
pnpm eval:hybrid-ab -- --knowledge-base-id=00000000-0000-4000-8000-000000000103 --dataset=olympus-zh --rerank=off --repetitions=3
pnpm eval:hybrid-ab -- --knowledge-base-id=00000000-0000-4000-8000-000000000103 --dataset=olympus-zh --rerank=on --repetitions=1
```
